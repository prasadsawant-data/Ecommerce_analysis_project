# 🛒 E-commerce Data Analysis Project

## 📊 Project Overview
This project focuses on analyzing e-commerce data to generate meaningful business insights. The analysis helps understand customer behavior, revenue trends, and product performance.

---

## 🎯 Objective
To analyze sales data and identify key factors affecting revenue, customer segmentation, and business growth.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL (MS SQL Server)
- Power BI
- Excel

---

## 📁 Dataset
- Olist E-commerce Dataset
- Includes customers, orders, payments, products, and reviews data

---

## 📊 Key Analysis Performed

### 🔹 SQL Analysis
- Total orders and revenue calculation  
- Orders by status  
- Monthly revenue trend  
- Top customers and cities  
- Delivery performance & late deliveries  

---

### 🔹 Python Analysis
- Data cleaning and preprocessing  
- Monthly revenue trend visualization  
- Top 5 cities by revenue  
- Customer segmentation (CLV analysis)  
- Order status vs revenue analysis  

---

### 🔹 Power BI Dashboard
- KPI Metrics:
  - Total Orders: 99.44K  
  - Average Order Value: 136.68  
  - Total Revenue: 13.59M  

- Visuals:
  - Top product categories  
  - Revenue by cities  
  - Monthly revenue trend  

---

## 📈 Key Insights
- Delivered orders generate the highest revenue  
- Top cities contribute major share of revenue  
- Revenue shows seasonal trends  
- VIP customers generate high revenue despite being fewer  
- Some delays in delivery impact customer experience  

---

## 🚀 Business Recommendations
- Focus on high-value (VIP) customers  
- Improve delivery performance  
- Target top-performing cities  
- Use seasonal trends for marketing strategies
